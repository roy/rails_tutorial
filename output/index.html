Het bouwen van een applicatie

Pizzeria applicatie

De applicatie dat we gaan bouwen is een pizzeria website met administratie backend. Op de website kan een bezoeker pizza's bestellen en als bonus kan een bezoeker de ingredienten kiezen voor een gehele eigen pizza. Tevens komt er een backend waar de eigenaar van de webshop pizza's kan definieren en bij kan houden hoeveel omzet de eigenaar heeft gemaakt
